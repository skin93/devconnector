# DevConnector
Social network for developers (MERN stack application). App deployed to Heroku.

## Built with
```
1. HTML5, CSS3, JavaScript ES6, Async/Await, Axios, JSON Web Token
2. React, Redux, React Hooks
3. Node.js, Express
4. MongoDB, mongoose
```
